# Laura Folk Fair - Website Design Document

## Project Overview
A professional brochure website for the Laura Folk Fair, a community event in Laura, South Australia. The site will be built with HTML, Tailwind CSS, and Google Fonts/Icons, following the modern, bold style of Aussie World theme park website.

## Design System & Style Guide

### Color Palette (Based on Aussie World reference)
- **Primary**: Bold, vibrant colors (to be determined from reference site analysis)
- **Secondary**: Complementary colors
- **Accent**: Highlight colors for CTAs
- **Neutral**: Grays for text and backgrounds

### Typography
- **Google Fonts**: 
  - Headings: Bold, modern sans-serif (e.g., "Poppins", "Montserrat", or similar)
  - Body: Clean, readable sans-serif (e.g., "Inter", "Open Sans", or similar)
- **Font sizes**: Responsive, mobile-first approach

### Icons
- **Google Material Icons** or **Font Awesome**
- Consistent icon usage throughout

### Layout Principles
- **Mobile-first responsive design**
- **Clean, modern aesthetic** inspired by Aussie World
- **Bold headlines and clear CTAs**
- **Generous white space**
- **Card-based layouts** for content sections
- **Hero sections** on each page

### Image Guidelines
- **Placeholder images** to be used initially
- **Alt text required** for all images (SEO requirement)
- **Optimized file sizes** for fast loading
- **Responsive images** (srcset for different screen sizes)

### SEO Elements (Per Page)

#### Meta Tags (Required for each page)
- **Title tag**: 50-60 characters, unique per page
- **Meta description**: 150-160 characters, unique per page
- **Open Graph tags** for social sharing
- **Canonical URL**

#### Heading Structure
- **H1**: One per page, main headline
- **H2**: Section headings
- **H3**: Subsection headings
- **Proper hierarchy**: H1 ‚Üí H2 ‚Üí H3

#### Content Requirements
- **Unique content** on each page
- **Keyword optimization** (natural, not keyword stuffing)
- **Internal linking** between related pages
- **Image alt text** for all images
- **Schema markup** (JSON-LD) for events, organization, local business
